S0: As a customer of the museum, I'd like to be able to point my phone camera at the artifact's description and see an artifact appear on my phone screen.

S1: As a customer admiring an artifact, I'd like to be able to manipulate it so I can view it from all perspectives.

S2: As the owner of the museum, I don't want people handling one of a kind ancient artifacts So nothing gets broken.

S3: As a curator of the museum I'd like to upload new artifacts to the app so that visitors will be able to appreciate them.

S4: As someone who can't physically go to the museum, I'd like to be able to see the artifacts so I can enjoy history from the comfort of my own home.

S5: As a museum curator, I'd like to get feedback on popular artifacts so I can improve the museum layout
